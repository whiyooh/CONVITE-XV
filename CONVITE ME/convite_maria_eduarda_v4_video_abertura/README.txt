# Convite Maria Eduarda — versão 4

Fluxo atualizado:
1. Tela inicial com aparência de convite impresso fechado.
2. Ao clicar em abrir, aparece a tela "Convite especial" com um convite ilustrativo.
3. Ao clicar em "Ver convite", entra uma tela de vídeo de abertura por 8 segundos.
4. Após o vídeo, aparece a tela principal com os dados do convite e os botões.

Arquivos:
- index.html
- style.css
- script.js
- assets/envelope-bg.png

Observações:
- As imagens dos cards e modais estão puxando URLs externas da web.
- O vídeo também está puxando uma URL externa.
- Para trocar o vídeo, altere o atributo src do elemento <video id="opening-video"> no index.html.
- Para usar imagens locais, coloque os arquivos na pasta assets e substitua as URLs no HTML/JS.
